# Utilisation CLI

- Lancer un backup immédiat :

```bash
node src/index.js
```

- Lancer le mode planifié (selon CRON_SCHEDULE) :

```bash
node src/index.js
```

- Tester la rétention :

```bash
node src/test-retention.js
```

- Tester la notification email :

```bash
node src/test-notify.js
```
